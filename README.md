# RDG_XP
RANDOM DUNGEON GEENERATOR (RDG) by XP
Das ist ein simples Dungeon Generator Tool, welches zufällig Dungeons generiert.
Es würde in C# geschrieben.

## Features
- Zufällige Dungeon Generierung
- Einstellbare Dungeon Größe
- Spieler bewegung
- Einfache Benutzeroberfläche

## Wie spielt man es?!
1. Kopire den Code von RDG.cs in eine C# Entwicklungsumgebung (z.B. Visual Studio).
2. Kompiliere und starte das Programm.
3. Viel Spaß beim Erkunden der Dungeons!
